## AoC 2023

In JavaScript and with [peggy.js](https://peggyjs.org/).

## Getting Started

### Prerequisites

- **Node.js** v18 or newer
- **pnpm**

### Installing and test

In your terminal,

```bash
$ pnpm install
$ pnpm test
```

## Structure

Solutions for each day are in `src/`. Their corresponding input are in `src/input/`.
